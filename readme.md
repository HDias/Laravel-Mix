# Started Project Laravel-Mix from Laravel 5.5

## Defaults Packages

 - bootstrap 4
 - jquery 3.3.1
 - axios 0.17
 - cross-env 5.1
 - laravel-mix 1.0
 - lodash 4.17.4
 - popper.js 1.12.9


 ## To RUN

 ```
 npm install

 // Run all Mix tasks...
 npm run dev

 // Run all Mix tasks and minify output...
 npm run production
 ```

 ## To Watch

 ```
 npm run watch
 ```

 More in [Laravel Docs 5.5](https://laravel.com/docs/5.5/mix)
